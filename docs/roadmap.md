# Roadmap

## v0.1 — local-first MVP

- Native Codex, Claude Code, and Grok CLI adapters
- Daily, weekly, monthly, and yearly TUI views
- Per-model uncached input, cached input, 5-minute cache write, 1-hour cache write, output, and cost
- JSON and non-interactive table output
- Windows and Linux CI/package validation

## v0.2 — faster and broader

- Incremental file index so large histories refresh in under one second
- CSV export and configurable currency display
- Live, signed pricing catalog updates with a reproducible offline fallback
- Additional adapters driven by redacted schema fixtures
- Exact Grok input/cache/output accounting if the local session format exposes it

## v1.0 — stable automation surface

- Versioned JSON output schema
- Stable adapter API
- Performance budgets and large-history fixtures
- Published npm package with provenance
